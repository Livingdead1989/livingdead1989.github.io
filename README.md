# NetworkingDream

**NetworkingDream** is a personal knowledge hub focused on networking, infrastructure, systems engineering, and experimentation.

🌐 **Live site:** https://networkingdream.com/

---

## 🧱 Stack

- **Static site generator:** Jekyll  
- **Hosting:** GitHub Pages  
- **Styling:** Vanilla CSS with CSS custom properties  
- **Icons:** SVG  
- **No frameworks** — intentionally lightweight and dependency-free

---

## 🚀 Local Development

```bash
bundle install  
bundle exec jekyll serve
```

or 

```bash
rake serve
```

Then open:  
http://localhost:4000/

## 📌 Status

This site is a **living project** — content, design, and structure evolve over time as ideas mature.
